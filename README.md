How to Play
SPACE: Perform an action.
q: Change tools.
e: Change seeds.
Enter: Go to sleep, open the shop.
Esc: Close the shop, exit the menu.
P: Pause the game to turn off music, choose a map, or quit the game.
M: Open the settings menu.
